# Conditional Statements in Python

Conditional statements are used in Python to perform decision-making operations. They allow a program to execute different blocks of code depending on whether a condition is true or false.

Conditional statements help control the flow of a program and make applications more interactive, logical, and dynamic.

---

# Importance of Conditional Statements

Conditional statements are important because they help programs:

- Make decisions
- Compare values
- Execute specific code blocks
- Control the flow of execution
- Handle different situations dynamically

Without conditional statements, programs would execute every line of code in the same order without any logic or decision-making ability.

---

# Types of Conditional Statements in Python

Python provides different types of conditional statements for handling various logical situations.

## 1. if Statement

The `if` statement executes a block of code only when the specified condition is true.

It is the most basic conditional statement in Python.

---

## 2. if-else Statement

The `if-else` statement is used when there are two possible outcomes.

If the condition is true, the `if` block executes. Otherwise, the `else` block executes.

This helps programs perform alternative actions when conditions fail.

---

## 3. if-elif-else Statement

The `elif` statement stands for "else if".

It is used to check multiple conditions one after another.

If one condition becomes true, the remaining conditions are skipped.

This structure is useful when programs need to handle multiple possible cases.

---

## 4. Nested if Statement

A nested `if` statement means placing one conditional statement inside another conditional statement.

Nested conditions are useful when multiple levels of checking are required.

This is commonly used in authentication systems, validations, and complex decision-making programs.

---

# Comparison Operators

Conditional statements use comparison operators to compare values.

These operators return either `True` or `False`.

| Operator | Meaning |
|----------|----------|
| == | Equal to |
| != | Not equal to |
| > | Greater than |
| < | Less than |
| >= | Greater than or equal to |
| <= | Less than or equal to |

Comparison operators are an essential part of writing conditions in Python.

---

# Logical Operators

Logical operators are used to combine multiple conditions together.

Python provides three main logical operators:

| Operator | Meaning |
|----------|----------|
| and | Returns true if both conditions are true |
| or | Returns true if at least one condition is true |
| not | Reverses the result of the condition |

Logical operators help create more advanced and complex conditions in programs.

---

# Indentation in Python

Python uses indentation to define blocks of code.

Proper indentation is extremely important because incorrect spacing can produce errors.

Every statement inside a conditional block must be properly indented.

Indentation improves code readability and structure.

---

# Flow of Execution

Conditional statements control how a program executes.

The program checks conditions one by one and executes only the matching block of code.

This process is called flow control.

Flow control helps programs behave differently based on user input, data, or situations.
