# Variables in Python

## What is a Variable?

A variable is used to store data.

Example:

```python
name = "Ayesha"
age = 18
```

Here:
- `name` stores text
- `age` stores a number

---

## Rules for Variables

✅ Can contain:
- letters
- numbers
- underscore (_)

❌ Cannot:
- start with a number
- contain spaces
- use special symbols

Correct:
```python
user_name = "Ali"
age1 = 20
```

Wrong:
```python
1name = "Ali"
user name = "Ali"
```

---

## Types of Variables

### String
```python
city = "Hyderabad"
```

### Integer
```python
age = 18
```

### Float
```python
height = 5.8
```

### Boolean
```python
is_student = True
```
